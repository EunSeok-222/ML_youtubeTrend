노트북 실행 순서

1. 01_eda.ipynb           — 데이터 로드·KR 필터링·탐색적 데이터 분석(EDA)
2. 02_preprocessing.ipynb — 파생 변수 생성·종속변수 정의·데이터 프로파일링
3. 03_modeling.ipynb      — 5개 모델 비교(RF·GB·XGBoost 등)·하이퍼파라미터 튜닝
4. 04_evaluation.ipynb    — 개선 모델 평가(GradientBoosting·XGBoost·ROC 곡선)

실행 전: pip install -r requirements.txt jupyter
데이터:  trending_yt_videos_113_countries.csv 파일을 프로젝트 루트에 위치시킬 것
